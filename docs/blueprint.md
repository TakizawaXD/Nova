# **App Name**: NovaSphere

## Core Features:

- User Authentication & Profile: Users can register and log in with email/password. Each user has an editable profile including a username, profile picture, and a short bio.
- Social Feed & Posts: Users can create posts containing text and a single image. The feed displays posts chronologically from followed users. Users can like and comment on posts.
- Real-time 1-on-1 Chat: Private text messaging between two users in real-time.
- Vertical Video Feed: A dedicated section to view short, vertical videos (pre-loaded content for MVP). Users can like and comment on these videos.
- Real-time Notifications: Instant notifications for new likes, comments on user's posts, and new messages.
- AI Post Discovery Tool: An AI tool that suggests popular posts from across the platform, dynamically updating based on engagement metrics to enhance content discovery.

## Style Guidelines:

- Color scheme is predominantly dark, aiming for a futuristic and elegant feel. The primary color is a vibrant electric violet-blue for interactive elements: HSL(250, 80%, 50%), converted to hex: #5E12E5.
- The background color maintains the primary hue but is heavily desaturated and dark, providing a subtle depth. HSL(250, 15%, 10%), converted to hex: #15131C.
- An analogous accent color, a bright celeste blue, is used for highlights, calls-to-action, and secondary interactive elements, ensuring strong contrast. HSL(220, 80%, 70%), converted to hex: #94A7FF.
- The font 'Inter' (sans-serif) is recommended for both headlines and body text due to its modern, clean, and highly legible characteristics, fitting the sleek and contemporary aesthetic.
- Icons should be clean, minimalist, and use an outlined style. Geometric shapes are preferred to maintain the futuristic and modern look, aligning with inspirations like Discord and X.
- The layout features a left sidebar, a top navigation bar, and a central main feed. The design incorporates glassmorphism effects, blurred backgrounds, transparent elements, subtly floating cards with soft shadows, and consistently rounded borders to achieve a premium and dynamic feel across desktop and mobile views.
- Smooth and fluid animations should be implemented for state transitions, hover effects on interactive elements, and content loading. These animations, utilizing technologies like Framer Motion, will enhance the visual responsiveness and overall user experience, contributing to the 'adictiva visualmente' goal.
